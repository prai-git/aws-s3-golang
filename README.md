

Example of using WAS S3 services with golang
============================================

Description
-----------

This is simple examle of listing all object of S3 bucket. 

Usage
------

NOTE: Provide the parameters at command line 

	* Region
	* Access Key ID
	* Secret Access Key 
	* Bucket
	

```
./aws_s3_iam_example Region KeyId AuthKey Bucket 	
```
